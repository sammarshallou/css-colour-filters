Convert colour to CSS filter

If you have icons in black (or some other colour) and you want to change them to a different colour in CSS, you can use the filter property. This script calculates the filter value for any desired colour.

These filters are usually accurate to within ±1 R, ±1, G, ±1 B of the intended value. Sometimes they can be slightly less accurate in Firefox because of differences in filter calculations. (I think the difference may be that Firefox rounds colours to whole values before applying the next filter in the chain and Chrome does not.) It might be possible to get more consistent behaviour by changing the optimisation logic.
